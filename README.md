# EV Analytics Dashboard Using Power BI

## Project Overview

This project focuses on analyzing global Electric Vehicle (EV) data using Power BI to generate meaningful business insights through interactive dashboards and visualizations. The dashboard helps analyze EV growth trends, regional performance, powertrain adoption, category-wise comparison, sales, stock, and time-series analysis.

---

## Features

* Interactive Power BI Dashboard
* KPI Cards and Dynamic Visualizations
* Regional & Category Analysis
* Powertrain Analysis (BEV, PHEV, FCEV)
* Parameter & Unit Analysis
* Time-Series & Comparative Analysis
* DAX Measures and Calculated Columns
* Modern Dark Theme Dashboard Design

---

## Dashboard Pages

1. Overview Dashboard
2. Regional & Category Trends
3. Powertrain & Mode Insights
4. Parameter & Unit Analysis
5. Time Series & Comparative Analysis

---

## Tools & Technologies Used

* Power BI
* DAX (Data Analysis Expressions)
* Power Query
* Data Visualization Techniques

---

## Key Insights

* Global EV adoption increased rapidly after 2020
* BEVs became the dominant and fastest-growing powertrain type
* Top regions contributed the highest EV sales and stock growth
* EV stock and registrations showed strong year-over-year growth
* Time-series analysis revealed continuous growth in EV-related metrics

---

## Recommendations

* Expand EV charging infrastructure in high-growth regions
* Promote BEV adoption through policy support
* Use real-time analytics for better planning and decision-making
* Enhance future analysis with predictive analytics and forecasting

---

## Conclusion

The project successfully transformed raw EV data into meaningful insights using Power BI dashboards, helping analyze EV growth trends, regional adoption, and powertrain performance through interactive visualizations and KPI analysis.
